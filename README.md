# 📦 LogiSmart AI - Enterprise Logistics Tracker

A sleek, serverless dashboard for high-performance logistics monitoring.

## 🚀 Features
- **Real-time Satellite Tracking:** Powered by Leaflet.js and OpenStreetMap.
- **Weather Intelligence:** Live API integration for climate-aware shipping.
- **Excel Export:** Generate shipment and vendor reports instantly.
- **Enterprise UI:** Modern Glassmorphism design using Bootstrap 5.
- **Automated Alerts:** EmailJS integration for client notifications.

## 🛠️ Tech Stack
- Frontend: HTML5, CSS3, JavaScript (ES6 Modules)
- Styling: Bootstrap 5, Animate.css, FontAwesome
- APIs: OpenWeatherMap, EmailJS
- Libraries: Leaflet.js, XLSX (SheetJS), Toastify
